Syfy - Angular

Syfy - Angular Client

Syfy is a web app developed using the MEAN stack. It provides registered users with information about Syfy movies, directors, and genres. Users can create an account, update their personal information, and add/remove movies from their favorites. It uses the existing server-side RESTful API that I created using the Node.js framework Express and the NoSQL database program MongoDB.

Built with Angular, Angular Material, MongoDB, Node.js

Meet App

Meet App

The goal of this project was to build a serverless, progressive web application with React using a test-driven development (TDD) technique. The application uses the Google Calendar API to fetch upcoming events and the serverless function is hosted by cloud provider AWS. The beginning phase of this project involved creating user stories for each of the features I would be implementing in the app. I then turned these into Given-When-Then statements that formed the basis of my TDD approach. The app uses the Recharts library for data visualisation.

Built with React, Google Calendar API, AWS
